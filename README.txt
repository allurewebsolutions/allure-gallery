== Description ==
The Allure Gallery plugin provides functionality for the native WordPress gallery to have one image on top with thumbnails underneath.

== Installation ==
1) Activate the plugin
2) Insert a native WordPress gallery on a page
3) Select \"yes\" from dropdown of gallery options to activate the functionality
